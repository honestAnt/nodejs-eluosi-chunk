# 俄罗斯方块游戏 🎮

一个使用纯 JavaScript 开发的经典俄罗斯方块游戏，具有现代化的UI设计和流畅的游戏体验。

## 🎯 游戏特性

### 核心功能
- ✅ **经典方块形状**：包含全部7种俄罗斯方块（I、J、L、O、S、T、Z）
- ✅ **智能消行机制**：自动检测并消除完整行
- ✅ **等级系统**：随着消除行数增加，游戏难度逐渐提升
- ✅ **实时统计**：分数、等级、消除行数实时显示
- ✅ **下一块预览**：提前查看下一个方块形状
- ✅ **游戏状态管理**：开始、暂停、重新开始功能

### 操作控制
| 按键 | 功能 |
|------|------|
| ← → | 左右移动方块 |
| ↓ | 加速下落 |
| ↑ | 顺时针旋转方块 |
| 空格键 | 暂停/继续游戏 |

### 游戏机制
- 🎯 **计分系统**：消除行数 × 100 × 当前等级
- 📈 **等级提升**：每消除10行升级一次
- ⚡ **速度递增**：等级越高，方块下落速度越快
- 🎲 **随机生成**：方块随机出现，增加游戏挑战性

## 🖥️ 技术实现

### 技术栈
- **前端**：HTML5 + CSS3 + Vanilla JavaScript
- **后端**：Node.js HTTP Server
- **图形渲染**：Canvas API
- **包管理**：npm / yarn

### 核心技术亮点
- 🎨 **Canvas 2D渲染**：使用原生Canvas API进行游戏画面绘制
- 🔄 **游戏循环**：基于`requestAnimationFrame`的流畅动画循环
- 🎯 **碰撞检测**：精确的方块边界和旋转碰撞检测算法
- 📱 **响应式设计**：适配桌面和移动设备
- 🎭 **现代UI**：玻璃质感设计和渐变色彩搭配

## 🚀 快速开始

### 环境要求
- Node.js 14.0.0+（推荐使用最新LTS版本）
- 现代浏览器（支持HTML5 Canvas）
- npm 或 yarn 包管理器

### 安装与运行

1. **克隆项目**
   ```bash
   git clone <repository-url>
   cd node-game-eluoshi
   ```

2. **安装依赖**
   ```bash
   npm install
   # 或使用 yarn
   yarn install
   ```

3. **启动服务器**
   ```bash
   # 生产模式
   npm start
   
   # 开发模式（自动重启）
   npm run dev
   ```

4. **访问游戏**
   打开浏览器，访问：`http://localhost:3000`

5. **开始游戏**
   点击"开始游戏"按钮即可开始！

## 🎮 游戏说明

### 游戏目标
- 控制下落的俄罗斯方块，填满水平行以消除它们
- 避免方块堆积到顶部导致游戏结束
- 尽可能获得更高的分数和等级

### 游戏规则
1. 方块从顶部自动下落
2. 使用方向键控制方块移动和旋转
3. 填满一整行时该行会被消除
4. 同时消除多行可获得更高分数
5. 当新方块无法放置时游戏结束

### 计分规则
- **单行消除**：100分 × 当前等级
- **多行消除**：按消除行数累加计分
- **等级提升**：每消除10行等级+1
- **速度增加**：等级越高，下落间隔越短
